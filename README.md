## Installation

Windows is currently not supported. Tested successfully on OS X Yosemite.

```bash
$ brew install mono
$ npm install -g dnvm
```

## Usage

```bash
$ dnvm upgrade
$ dnu restore
$ dnu build
$ dnx . kestrel
````
